# usd cash calculator

Aplicación web para calcular cantidades de dinero en dólares estadounidenses ingresando la cantidad de billetes y monedas.

## características

- cálculo automático en tiempo real
- soporte para billetes y monedas USD
- interfaz simple y responsive
- actualización instantánea sin botón de calcular

## tecnologías

- html
- css
- javascript

## funcionamiento

El usuario ingresa la cantidad de cada denominación y el sistema calcula automáticamente el total acumulado.
